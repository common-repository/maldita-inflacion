Cansado de cambiar los precios una y otra vez por la inflacion? Este plugin actualiza los precios en toda la tienda de acuerdo al valor del dolar blue automaticamente. Tambien puede usar el dolar oficial o un valor asignado
manualmente

== Description ==
Actualiza manual o automaticamente los precios de los productos WooCommerce. 

Este plugin no ejecuta ningun tipo de accion en el frontend.

Los precios son editados automaticamente desde la base de datos. 

Puedes utilizar el tipo de cambio oficial o paralelo provenientes de la API. 

Tambien es posible asignar un valor para el tipo de cambio manualmente.

Las actualizaciones de precios se realizan de forma automatica de acuerdo a la cantidad de dias seleccionados. 

La actualizacion de precios se ejecuta unicamente al cargar cualquier pagina del backend, 

Esta es la primer version del plugin, toda sugerencia, opinion o critica sera bienvenida.

== Frequently Asked Questions == 
=Que pasa con los precios si desactivo el plugin=
Los precios son actualizados desde la base de datos, por lo que al desactivar o desinstalar el plugin nada cambiara

=Como vuelvo a ingresar los precios en mis productos al desactivar el plugin?=
Simplemente utiliza el campo "precio" en la pagina de edicion de producto, como lo harias normalmente. Si el plugin sigue activo, asegurate de que el campo "valor TDC" se encuentra vacio
