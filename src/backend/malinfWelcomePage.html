MALDITA INFLACION !

Convierte los precios en tu tienda en base al cambio dolar oficial/blue

Actualiza automaticamente en base a respuesta api

Esta API es de uso gratuito y es de donde extrae los datos este plugin.

Opciones de la pagina Ajustes:

1_ API

Selecciona la opcion modo API para actualizar los precios automaticamente

Selecciona el periodo de tiempo entre cada actualizacion de precios

Seleccione el tipo de cambio dolar oficial o blue


2_ MANUAL

Selecciona la opcion modo MANUAL para ingresar el valor de cambio manualmente

Ingresa un valor numerico


3_ Redondeo

Redondea los valores al realizar la actualizacion de precios

  1. -No redondear: redondeo desactivado
  2. -Abajo: redondea hacia abajo de acuerdo al alcance asignado
  3. -Arriba: redondea hacia arriba de acuerdo al alcance asignado
  4. -Alcance: rango en el que aplicar el redondeo

    Ejemplo (Redondeo: abajo - Alcance: 10)

    2348 = 2340

    Ejemplo (Redondeo: arriba - Alcance: 10)

    2348 = 2350

    Ejemplo (Redondeo: abajo - Alcance: 100)

    2348 = 2300

    Ejemplo (Redondeo: arriba - Alcance: 100)

    2348 = 2400


4_ GUARDAR

Guarda los cambios de las opciones parra la proxima actualizacion


5_ CAMBIAR AHORA

Convierte los precios de tus productos a valor de cambio asignado pulsando el boton CAMBIAR AHORA

  1. Al presionar el boton, se actualizaran los precios de todos los productos que tengan un valor de cambio asignado

  2. Se registra la accion para efectuarse a la misma hora, en base al rango temporal de actualizacion seleccionado en dias



Opciones de la pagina Configuracion:

Convertir precios a valor de TDC

Si los precios estan expresados en dolares, convierte a pesos asignando el precio actual al valor del Tipo de cambio

No ejecuta ninguna actualizacion de precios en la tienda, solo guarda los precios como valor de Tipo de Cambio para cada producto


Limpiar base de datos

Elimina registros del plugin al desinstalarlo

Modo de uso:

Al activar el plugin por primera vez, se vera un campo adicional en la pagina de edicion de productos.

Valor TDC (Tipo De Cambio)

Para comenzar, es necesario ingresar un valor en este campo.

El valor del tipo de cambio que se aplicara en cada actualizacion de precios puede ser una de las siguientes segun la configuracion de las opciones:

  1. Modo API: dolar oficial o blue

  2. Modo Manual: El valor ingresado en la opcion "Valor del tipo de cambio

Repetir la operacion en todos los productos a los que se desea aplicar la conversion con el tipo de cambio

No es necesario ingresar el valor en todos los productos. Los que no tengan valor TDC no se actualizaran

Una vez finalizado el proceso pulsar el boton CAMBIAR AHORA para realizar las actualizaciones de precio en toda la tienda

Solo los productos que tengan asignado un valor en el campo VALOR TIPO DE CAMBIO seran actualizados

Para desactivar la actualizacion en productos individuales, dejar el campo valor TDC vacio

Una vez hecho lo anterior, puedes actualizar el precio nuevamente usando el campo "precio" de WC como lo harias normalmente al editar un producto

Funcionamiento:

• La actualizacion de precios se realiza pulsando el boton Cambiar Ahora

• En ese mismo momento, se iniciara el cronometro para llegar a la proxima actualizacion

• La actualizacion de precios se realiza automaticamente al cargar cualquier pagina del backend

• Las actualizaciones no se efectuan en segundo plano o sin carga del backend

• Este plugin no ejecuta ningun tipo de accion en el frontend

• El rango de tiempo entre cada actualizacion esta dado en dias pero no se ajusta al formato fecha (30 dias puede no ser un mes exactamente)

• El campo de producto precio de WooCommerce queda desactivado si el campo Valor TDC no esta vacio

No soporta precio de oferta "sale"

No soporta productos variables
